# 小学生朗读激励抽奖应用

一个本地 Windows 桌面应用，用于通过麦克风检测学生朗读音量，驱动"抽奖结果逐渐揭晓"的互动效果，以提升小学生阅读兴趣。

## 快速开始

### 环境要求

- Python 3.10 或更高版本
- Windows 操作系统

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
python app.py
```

## Phase 1 功能说明

### 配置界面功能

1. **麦克风设置**
   - 选择麦克风设备（当前为模拟数据）
   - 注：实时音量测试将在 Phase 2 实现

2. **活动参数设置**
   - 音量阈值：设置触发朗读的音量大小（0.01-1.0）
   - 揭晓时长：设置达到目标所需的持续时间（秒）
   - 回退时长：设置音量不足时的回退速度（秒）

3. **奖项设置**
   - 添加奖项：点击"➕ 添加奖项"按钮
   - 编辑奖项：直接在表格中修改名称和权重
   - 删除奖项：点击对应行的"删除"按钮
   - 至少需要保留一个奖项

4. **配置管理**
   - 保存配置：将当前设置保存到 `data/config.json`
   - 自动加载：启动时自动加载上次保存的配置
   - 验证机制：保存前会验证配置的合法性

5. **开始活动**
   - 点击"开始活动"按钮会验证配置
   - 当前会显示配置摘要（实际活动界面将在 Phase 3 实现）
